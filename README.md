# edicao-grafica-vue

## Project setup
```
npm install -g @vue/cli
```

## Check vue version
```
vue --version
```

## Update vue version
```
npm update -g @vue/cli
```

## Create project
```
vue create 'project-name'
```

## Or create project via UI
```
vue ui
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
